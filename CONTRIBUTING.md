# Contributing

This is a small public reference. Contributions are welcome if they keep the
library RPC-free and keep the builder unsigned.

## Local dev

```
npm install
npm run build
npm test
```

## What this repo is not

- Not a production service.
- Not affiliated with any running deployment.
- Not a place to sign transactions on behalf of users.

If your change pulls the library in any of those directions, please open an
issue first and let's talk it through.
